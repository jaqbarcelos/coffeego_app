CoffeeGo é um aplicativo desenvolvido para conectar amantes de café às melhores
cafeterias da região. A plataforma permite localizar estabelecimentos, visualizar cardápios,
favoritar cafeterias e itens do cardápio, realizar pedidos e acompanhar o histórico de
compras, proporcionando praticidade e uma experiência agradável ao usuário.

Para desenvolver o APP, foi utilizado o mockAPI, onde ficou registrado os produtos e os
dados dos usuários.
