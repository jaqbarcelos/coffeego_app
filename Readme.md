# ☕ CoffeeGo — Aplicativo de Delivery de Café

Aplicativo mobile de delivery de café desenvolvido com **React Native** e **Expo**, como projeto acadêmico da disciplina de Programação para Dispositivos Móveis.

## 📱 Sobre o projeto

O CoffeeGo é uma solução digital para uma rede de cafeterias, permitindo que clientes:
- Naveguem pelo cardápio com filtros por categoria
- Façam pedidos com entrega ou retirada
- Acompanhem o pedido em tempo real
- Gerenciem seu perfil e endereços
- Salvem cafeterias e produtos favoritos

## 🚀 Tecnologias utilizadas

- **React Native** — desenvolvimento mobile multiplataforma
- **Expo Snack** — ambiente de desenvolvimento online
- **MockAPI** — banco de dados REST na nuvem
- **Context API** — gerenciamento de estado global
- **React Navigation** — navegação entre telas
- **Animated API** — animações nativas

## 🗂️ Estrutura do projeto

- context/        # Context API (estado global)
- navigation/     # Navegação e Drawer customizado
- screens/        # Telas do aplicativo (14 telas)
- components/     # Componentes reutilizáveis
- theme/          # Cores e estilos globais

## ⚙️ Como rodar o projeto

1. Acesse o link do Expo Snack abaixo
2. Clique em "Run" para executar no emulador web
3. Ou escaneie o QR Code com o app **Expo Go** no celular

🔗 **Link do projeto no Snack:** https://snack.expo.dev/@jaquebarcelos/app_coffeego
